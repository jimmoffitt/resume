# Resume

Digital resume/CV, published via GitHub Pages.

## Contents

- `index.html` — single-file resume (HTML/CSS, no build step, no dependencies beyond Google Fonts)

## Updating

1. Edit `index.html` directly (or regenerate it and re-upload).
2. Commit to `main`.
3. GitHub Pages redeploys automatically within a minute or two — no further action needed.
